Utilisation de la caméra

Pdfigo utilise la caméra de votre appareil uniquement pour la fonctionnalité de scanner de documents.

  • Les photos sont traitées localement sur votre appareil
  • Aucune image n'est envoyée à des serveurs tiers
  • Aucune image n'est stockée en dehors de votre appareil sans votre accord
  • Vous pouvez refuser l'accès à la caméra dans les paramètres Android
Fonctionnalités IA (Résumé, Traduction, Chat PDF)

Lorsque vous utilisez les fonctionnalités d'intelligence artificielle, le texte extrait de votre document est envoyé de manière sécurisée (HTTPS) à nos serveurs pour traitement.

⚡ Le texte est transmis à Google Gemini ou Mistral AI via notre infrastructure sécurisée (Supabase). Aucun contenu n'est stocké après traitement. Vos fichiers PDF originaux ne quittent jamais votre appareil.

En utilisant ces fonctionnalités, vous acceptez les politiques de confidentialité de Google et Mistral AI.

Abonnements et paiements

Les paiements sont entièrement gérés par Google Play. Pdfigo n'a jamais accès à vos informations bancaires ou de carte de crédit.

Nous utilisons RevenueCat pour gérer les statuts d'abonnement. RevenueCat stocke un identifiant anonyme lié à votre compte Google Play. Consultez la politique de confidentialité de RevenueCat.

  • Essai gratuit de 14 jours pour les nouveaux utilisateurs
  • Renouvellement automatique, annulable à tout moment
  • Remboursement selon la politique Google Play

Description du service

Pdfigo est une application mobile de gestion de documents PDF proposant des fonctionnalités d'édition, de signature, de compression, de protection, d'OCR et d'intelligence artificielle.

L'application est disponible en version gratuite (fonctionnalités limitées) et en version Premium (accès illimité via abonnement).

Contrôle de vos données

Vous pouvez contrôler et supprimer les données stockées localement :

  • Préférences app : Paramètres Android → Applications → Pdfigo → Effacer les données
  • Fichiers PDF : Depuis l'écran "Mes Fichiers" dans l'application
  • Code PIN : Depuis les Paramètres de sécurité dans l'application
  • Abonnement : Google Play → Abonnements → Pdfigo → Annuler
  • Données RevenueCat : Contactez-nous à devbp2025@gmail.com
